Andon Floor Unit OTA Update Tool

Overview

Developed a secure and automated OTA update system for manufacturing floor Andon units, enabling reliable deployment of software updates at scale.

System Design

  • SSH-based remote deployment architecture
  • Cron-driven updates pulling from GitLab via service accounts
  • Raspberry Pi 4 base station hosting update binaries
  • HTTP server distribution with automated flash & service restart

Key Contributions

  • Designed fault-tolerant update workflow for industrial environment
  • Automated version distribution and rollout
  • Eliminated manual update steps across floor units

Impact

Improved deployment reliability while dramatically reducing maintenance effort and system downtime.
