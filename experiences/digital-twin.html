Factory Floor Digital Twin

Overview

Developed a real-time digital twin of the manufacturing floor to visualize equipment health, production metrics, and operational status across multiple workstations.

System Architecture

  • IoT data ingestion using NCD.io industrial sensors
  • MQTT-based messaging for real-time data transport
  • Node-RED pipelines for data processing and routing
  • ThingsBoard dashboards for visualization and analytics

Key Contributions

  • Designed and deployed end-to-end industrial IoT data flow
  • Integrated machine health and production telemetry
  • Created real-time dashboards for operations visibility
  • Enabled proactive maintenance and downtime identification

Impact

Provided production teams with live insight into factory operations, improving situational awareness and decision-making.
