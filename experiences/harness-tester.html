Universal Harness Continuity & Crimp Test System

Overview

Designed and implemented a universal electrical harness testing system to validate continuity and crimp integrity across full product harness lineup. The system significantly improved test coverage, repeatability, and manufacturing efficiency.

System Description

  • Arduino Mega as primary controller
  • Integrated LCD display for technician interaction
  • Dual-sided chassis architecture
  • Support for all Molex and JST connectors used across products
  • Support for Deutsch connectors ranging from 4-pole to 8-pole

Key Contributions

  • Designed scalable harness test architecture covering all connector variants
  • Implemented automated continuity and crimp validation logic
  • Developed LCD-driven UI for guided test workflows
  • Reduced operator error through standardized test procedures

Impact

Improved harness validation reliability while reducing manual inspection time and operator dependency during production testing.
