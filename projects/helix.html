Helix – Autonomous Mobile Robot

Overview

Helix is a small autonomous UGV designed for outdoor navigation using LiDAR-based mapping and embedded motor control.

System Architecture

  • Raspberry Pi 4B – ROS2, SLAM, planning
  • Teensy 4.1 – real-time motor control and IO
  • UART communication between controllers

Key Challenges

  • Deterministic motor control under ROS2 command velocity
  • Encoder-based odometry accuracy
  • Power protection for mixed-voltage system

Results

Achieved closed-loop velocity control and real-time odometry streaming to ROS2.

Future Improvements

Integrate IMU fusion using EKF and add autonomous path planning.
